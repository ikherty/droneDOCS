# FPV Glossary

This glossary explains common Russian FPV slang terms and abbreviations used in the FPV drone community.

| Term (Russian) | Meaning (English) |
| ------------------------------------- | ------- |
| ЛА (LA) | aircraft / flying vehicle |
| ВМГ (VMG) | propeller-motor group (motor + propeller assembly) |
| 2", 2.5", 3", 3.5", 4", 5", 6", 7" .. | propeller diameter, used to designate drone size classes |
| 31mm, 35mm, 40mm, 45mm | propeller diameter smaller than 2", mainly used among tiny whoops |
| tiny whoop, тинивуп, вуп (tinywhoop, whoop) | tiny copter with 65-75mm diagonal motor-to-motor distance, prop guards on frame, powered by 1-2S battery |
| синик, синевуп (cinewhoop) | from "cinematic" - copter for smooth flights to capture cinematic footage, with prop guards/ducts |
| зубочистка (toothpick) | very lightweight copter |
| батка, бата, АКБ, аккум, батарея (battery) | battery |
| 4ы, 4s (4S) | number of battery cells, in this example 4; each cell has 4.2V maximum voltage for LiPo/LiOn, or 4.35V for LiHV |
| LiPo, LiOn/LiIon, липо, лион | most common types of lithium batteries |
| LiHV, лихв, хв батка | lithium high voltage, battery with increased voltage, max 4.35V per cell, mainly used for tiny whoops |
| ирл, IRL | In Real Life |
| FPV | First Person View |
| OSD | On Screen Display |
| KV | RPM per volt (motor constant) |
| TVL | TV Lines - analog camera resolution specification |
| сетап, билд, сборка (setup, build) | refers to a complete copter build |
| стек, стак (stack) | tower of 4-in-1 ESC with flight controller, sometimes with video transmitter |
| канопа (canopy) | upper frame cover, typically plastic, protecting electronics |
| маунт (mount) | mounting hardware for camera, antenna, etc. |
| tx | transmitter, sends signal (can be a separate device or a transmitting pin on a board) |
| rx | receiver, receives signal (can be a separate device or a receiving pin on a board) |
| втх, vtx | video transmitter |
| рег, esc | ESC (Electronic Speed Controller) |
| приемыш (receiver) | radio receiver (slang) |
| передаст (transmitter) | radio transmitter (slang) |
| пк, мозг (FC, brain) | flight controller |
| редиска (radish) | a bad person (Russian slang, not really FPV-specific) |
| камень (stone) | microprocessor, STM chip |
| кондер (cap) | capacitor |
| резюк (resistor) | resistor |
| проп (prop) | propeller |
| аппа (radio) | radio transmitter / controller |
| бут, boot, дфу, дфю (boot, DFU) | bootloader mode, typically activated by holding the boot button while powering on |
| клипса (c-clip) | c-clip / e-clip |
| пантилт, pan-tilt | pan-tilt system for antennas, etc. |
| диверсити (diversity) | two receivers to output video from the source with better signal |
| кубик (cube) | action camera |
| серва (servo) | servo motor |
| пвм, pwm | PWM (Pulse Width Modulation) |
| ппм, сбас (ppm, sbus) | radio control protocols |
| аккст, аксес (ACCESS, ACCST) | FrSky protocol variants |
| бинд (bind) | process of linking a radio receiver to a transmitter; also applies to digital video systems |
| елрс, кросс, трейсер, гост (ELRS, Crossfire, Tracer, Ghost) | radio control systems, more info: https://propwashservice.com/intro/tx.html |
| ксв (SWR) | Standing Wave Ratio - ratio of maximum to minimum voltage amplitude in a transmission line |
| бетка (Betaflight) | Betaflight firmware |
| ченл мап (channel map) | channel order field in Betaflight configurator's receiver section; abbreviation made from first letters of channel names in the transmitter's mixes |
| акро, стаб, горайзн (acro, stab, horizon) | flight stabilization modes |
| фс, fail safe, фейлсейф, фэйлсэйф | drone actions on signal loss: drop (motor shutoff), land (landing - rarely used), GPS rescue (return home with auto-landing from BF 4.4+); in INAV: RTH (return to home) |
| таргет (target) | device configuration file used for firmware compilation |
| пассфру, пассру, пастру, пасстру (passthrough) | passthrough - mode for transmitting serial data to a device connected to the FC's UART/SoftSerial port |
| десинк, срыв синхры (desync) | situation when ESC loses track of motor coil commutation, causing the motor to stop; usually happens with rapid RPM changes; sometimes fixable via ESC settings |
| тайминги (timing) | angle at which the magnet should lag when energizing the coil - related to brushless motor operation |
| питч, ролл, яв, тротл (pitch, roll, yaw, throttle) | axis names controlled by the sticks |
| пагода, патч, сосиска (pagoda, patch, sausage) | types of FPV antennas |
| рилка (Reelsteady) | Reelsteady GO - software stabilization for GoPro video |
| накед, голая проха (naked GoPro) | "naked" GoPro - stripped-down GoPro Hero camera, weight reduced from 120g to 26g, powered by 5V BEC instead of battery |
| софтмаунт, демпфер (softmount, damper) | vibration isolation mount |
| кз, коза (short) | short circuit |
| синий дым (magic blue smoke) | the magic that makes all electronics work; when it escapes, something burns out |
| спинер жижа (conformal coating) | Spinner protection silicone conformal coating, based on MG CHEMICALS 422B |
| т12, тс100 (T12, TS100) | popular soldering iron models with T12 and TS tips respectively |
| редитускай (Readytosky) | Chinese brand that typically makes clones of known brands with their own logo |
| колобок (kolobok) | frame from Aerodyne-RC (also nickname of the author in some circles) |
| пигтейл (pigtail) | coaxial cable adapter |
| армиться, арм (arm) | mode to start motors at minimum throttle (throttle must be at bottom); without arming, the drone won't fly |
| дизармиться, дизарм (disarm) | turn off motors regardless of throttle level |
| джи, джедай, цифра (DJI, digital) | DJI digital FPV system |
| толстяк (fattie) | DJI FPV Drone |
| волки, улитка, аватар (Walksnail, Avatar) | Walksnail Avatar digital video system (a division of Caddx FPV) |
